# Visualize a data set or object

Synonyms of the
[`generics::visualize()`](https://generics.r-lib.org/reference/visualize.html)
method. Alternatives are `vis()` and `visualise()`.

## Usage

``` r
vis(x, ...)

visualise(x, ...)
```

## Arguments

- x:

  A data frame or other object.

- ...:

  Other arguments passed to methods
